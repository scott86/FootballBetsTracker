Setup
----------
1) Compile all .java files
2) Run java BetTrackerGUI

The first time it runs it will take a while to gather data records for all the season's games


Usage
---------
If running for the first time, click "save" to save found game records.
All data is saved in serialized Ticket objects saved in the working directory

1) Click the "bets..." button to add bets to the week's ticket.
2) Click the "save" button to save added bets to the week's ticket.
3) Keep the program running while the games are being played to see live updates
